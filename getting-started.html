    Game Engines

    Game Engines are the programs you use to program and help you render versions of your games. They come preloaded with a bunch of helpful shortcuts, tools and optimizations that make creating games easier.

    Game Genres

    Once you know what engine you're going to use for your game it can be helpful to understand what genres of games there are. Genre's act as a blanket theme or playstyle for your game. It helps give developers some direction in programing a game by having common tropes within games that apply to that genre specifically, and helps players know if your game is something they'd be interested in based off of genres they like.

    Starting Development

    Now that you have a vision for what game you want to create and have decided on the engine you're using its time to code! From basic character controls to make a game feel good and smooth to play, to enemy A.I. and basic level design this is where the fun begins!
